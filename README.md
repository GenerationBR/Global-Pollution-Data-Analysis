# Global Pollution Data Analysis (2025-2026)

## 📌 Project Overview
This project integrates data from multiple sources to analyze global pollution trends between 2025 and 2026. It covers the entire data pipeline, from raw data extraction to the creation of interactive dashboards, ensuring that business rules are correctly applied to the technical datasets.

## 🛠️ Tech Stack & Tools
*   **Language:** Python
*   **Data Manipulation:** Pandas, NumPy
*   **Environment:** Jupyter Notebook
*   **Data Visualization:** Looker Studio
*   **Process:** ETL (Extract, Transform, Load)

## 🚀 Key Features
*   **Data Extraction & Loading:** Efficient handling of large-scale pollution datasets.
*   **Data Cleaning:** Treatment of missing values, normalization, and data type consistency using Python.
*   **Business Logic Application:** Transforming raw metrics into actionable insights based on environmental benchmarks.
*   **Interactive Dashboards:** Visual reporting developed in Looker Studio for clear trend identification.

## 📊 Methodology
1.  **Exploration:** Initial analysis using **Jupyter Notebook** to understand data distribution.
2.  **Processing:** Leveraging **NumPy** and **Pandas** for high-performance data cleaning and filtering.
3.  **Validation:** Ensuring all data aligns with the project's specific business requirements.
4.  **Visualization:** Exporting processed data to **Looker Studio** to build intuitive and professional charts.
